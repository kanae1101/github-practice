## 学んだこと

while文は「条件がTrueの間繰り返す」文法である。

for文は回数や対象が決まっている繰り返し、
while文は条件による繰り返しであり、用途が異なることが分かった。

また、変数を更新しないと無限ループになることも学んだ。

## 辞書について

辞書は

"名前" → 値

の対応表を作ることができる。

例えば

"数学" → 95

のような形でデータを管理できる。

リストが番号で取り出すのに対し、辞書は名前で取り出すことができる。

## 印象に残ったこと

関数は実行結果だけでなく、関数そのものを変数や辞書に入れられることを知った。

calc_total

は関数そのものであり、

calc_total(scores)

は関数を実行した結果である。

そのため、

tools = {
"合計": calc_total,
"平均": calc_average
}

のような辞書を作ることができる。
