# 2026/6/24-25 学習記録

## 学んだこと

- while文を学習

  - 条件がTrueの間繰り返し処理を行う
  - 繰り返しに使う変数を更新しないと無限ループになる
  - 無限ループ時は Ctrl + C で停止できる

- 辞書（dictionary）を学習

  - キーと値の対応付け
  - `in` を使ったキーの存在確認
  - `keys()`、`values()`、`items()` の使い方
  - 関数を辞書に格納できることを知った

- 関数について理解を深めた

  - `calc_total` のような関数を変数として扱える
  - `tool = tools[tool_input]`
  - `result = tool(scores)`
  - `()` を付けると実行、付けないと関数そのもの

- 関数から複数の値を返す方法を学習

  - `return border, count`
  - `border, count = result`
  - 複数の値をまとめて扱える
