## 作成したプログラム

### BMI計算機

- 身長・体重からBMIを計算
- 小数第2位まで表示
- 判定（低体重・標準・肥満）付き

### シーザー暗号

- 大文字・小文字どちらにも対応
- 任意の文字数ずらせる
- アルファベット以外はそのまま出力

### 点数計算ツール
- 複数の点数を入力すると計算ができる
- 平均、最大値、中央値、最頻値、分散などを計算可能
